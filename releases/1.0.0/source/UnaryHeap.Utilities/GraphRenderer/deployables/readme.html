GraphRenderer

GraphRenderer is a command-line utility which can be used to convert JSON-serialized Graph2D objects into SVG files.

To use it, run it from the command prompt, specifying the input JSON file name as the first parameter and (optionally) specifying the output SVG file name as the second parameter. If the second parameter is omitted, the output file name will be that of the input file name with extension 'svg'.

Check out the examples subdirectory to see the expected JSON format and a batch file specifying the usage.

The latest version of this utility can be found at CodePlex.
